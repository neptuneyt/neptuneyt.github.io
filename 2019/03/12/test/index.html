awk版本

$ awk --version

GNU Awk 4.0.2
Copyright (C) 1989, 1991-2012 Free Software Foundation.

$ cat test.txt

1e2
1e-2
1e-10
1e-20
2e-10

$ awk '$1<1e-10' test.txt

1e-20

$ awk '$1>1e-4' test.txt

1e2
1e-2

$ awk '$1>0.01' test.txt

1e2

排序

$ sort -g  test.txt

1e-20
1e-10
2e-10
1e-2
1e2

$ sort -gr  test.txt

1e2
1e-2
2e-10
1e-10
1e-20
